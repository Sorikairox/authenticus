## authenticus

### Setup

#### Deno

```ts
import { GitHub } from 'https://deno.land/x/authenticus@v0.1.0/mod.ts'
```

#### Node.js

```bash
npm i authenticus
```

```ts
import { GitHub } from 'authenticus'
```

### Usage

```ts
const loginUrl = GitHub.login()
```
